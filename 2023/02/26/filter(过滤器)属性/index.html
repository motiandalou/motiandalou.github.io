filter(过滤器)属性

2023-02-26 CSS 971 字

前言: filter属性 我在日常开发用的并不多,但是我觉得很管用,有奇效,在一些特殊的样式上,你会觉得还可以这么简单就能实现.

  1. 定义: filter 属性定义了元素(通常是img标签)的可视效果(例如：模糊与饱和度)。
  2. 语法: filter: none | blur() | brightness() | contrast() | drop-shadow() | grayscale() | hue-rotate() | invert() | opacity() | saturate() | sepia() | url();
  1. blur 高斯模糊 (常用)
  2. brightness 亮度 (常用)
  3. contrast 对比度 (常用)
  4. drop-shadow 阴影 (重点)
  5. grayscale 灰度 (常用)
  6. hue-rotate 应用色相 (不常用)
  7. invert 反转输入图像 (不常用)
  8. opacity 透明度 (常用)
  9. saturate 图像饱和度 (不常用)
  10. sepia 将图像转换为深褐色 (常用)
  11. url (不用)

blur

语法: filter: blur(4px);
  1. 模糊度。默认为0, 值(px,不能写百分比)越大,越模糊.

brightness 

语法: filter: brightness(200%);
  1. 亮度。默认为1/100%, 图片没有变化,0是图片全黑,1是默认值颜色不变,值越大,图片越亮(即发白).

contrast 

语法: contrast(200%);
  1. 对比度。默认为1/100%, 图片没有变化,0是图片全黑,1是默认值颜色不变,值越大,对比度(即绿色更绿.黄色更黄).

drop-shadow 重点

语法: filter: drop-shadow(8px 8px 10px red);
  1. 第一个参数(必填): X轴水平位置(8px,即阴影向右移8px,负值相反)
  2. 第二个参数(必填): y轴水平位置(8px,即阴影向上移8px,负值相反)
  3. 第三个参数(选填): 模糊度(默认为0,值需要大于0),值越大,越模糊
  4. 第四个参数(选填): 阴影颜色

ps:与box-shadow的区别: inset + spread + 阴影叠加

语法: box-shadow: (inset,8px 8px 10px spread red);
  1. inset: 不填写,默认是向外阴影,设置inset(即内部阴影)
  2. spread: 默认值0(阴影和元素尺寸一致),控制阴影的范围扩大(值 > 0,值越大,阴影越大)或者缩小(值 < 0,阴影搜索)
  3. 阴影叠加: box-shadow值可以用逗号隔开,第一个层级(z-index)最高(8px(即阴影半径)),其次16px,最后24px.我们可以理解成一个圆心的靶子.
box-shadow: 0 0 0 8px #f0f, 
			0 0 0 16px #d00, 
			0 0 0 24pxpx #cdd,

grayscale 

语法: filter: grayscale(50%);
  1. 灰度。默认为0(图像无变化),最大值100%(完全变灰色).

hue-rotate

语法: filter: hue-rotate(90deg);
  1. 给图像应用色相旋转。默认为0deg(图像没有变化),一圈(0deg-360deg)效果循环,基本找不出规律(除非你对颜色很有研究),随着值的变化,你可以看到,粉色,紫色,棕色,粉红色,橙色..各种美丽的颜色.

invert

语法: filter: invert(100%);
  1. 反转输入图像。默认为0(图像没有变化), 50%(灰色), 100%(医院拍CT的片子一样).

opacity

语法: filter: opacity(30%);
  1. 透明度。默认为0(图像没有变化), 最大值100%(完全透明),值越大,透明度越高.

saturate

语法: filter: saturate(800%);
  1. 转换图像饱和度。默认为100%(图像无变化), 0(完全不饱和),值越大,饱和度越大(即颜色越鲜艳).

sepia

语法: filter: sepia(100%);
  1. 将图像转换为深褐色。默认为0(图像无变化),最大值100%(完全变深褐色).

url (没用过,不知道退役之前能不能用上)

语法: filter: url(svg-url#element-id)
  1. 默认为0, 值(px,不能写百分比)越大,越模糊.

日常前端开发需要注意的点

  1. grayscale,可以用在网页一些特殊日子,网页全部变成灰色
  2. drop-shadow 可以与box-shadow 可以按需求,交替/对比使用,日常开发中,会有意想不到的效果.
  3. opacity(所有元素都具有透明度)/rgba(背景出现透明效果，但上面的文字不透明)进行设置透明度,可以按需求,交替/对比使用.
  4. sepia可以做成有年代感照片的效果.
